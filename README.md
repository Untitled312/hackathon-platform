# Hackathon Platform

Веб-приложение для организации хакатонов

# Запуск

1. Скачайте и распакуйте zip-архив

2. Перейдите в папку hackathon-platform-main

3. Откройте два терминала:

```
cd frontend
npm install
npm run dev
```

```
cd backend
dotnet run
```

4. Перейдите по адресу http://localhost:5173
